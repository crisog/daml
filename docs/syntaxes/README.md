# Daml syntax grammar

This directory contains the Daml TextMate grammar used for code highlighting in the docs site.

- `daml.json` is vendored for deployment reliability (no submodule checkout required at build time).
- Source upstream: `external/daml/sdk/compiler/daml-extension/syntaxes/daml.json`

When updating the grammar, copy the upstream file and keep this note in sync.
